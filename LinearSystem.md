## 問6
```
h[n] = if n = 1,2: 2 else: 0
x[n] = if n = -2,1: 2 else: 0
```
δの式に直す
```
h[n]=2δ[n-1]+2δ[n-2]
x[n]=2δ[n+2]+2δ[n+1]
```
入力```δ[n]```,出力```2δ[n-1]+2δ[n-2]```  
LTIシステムなので  
入力```2δ[n]```,出力```4δ[n-1]+4δ[n-2]```  
入力```2δ[n+2]```,出力```4δ[n+1]+4δ[n]```  
  
入力```2δ[n+1]```,出力```4δ[n]+4δ[n-1]```  
    
```
y[n]=PI{ 2δ[n+2]+2δ[n+1] }
    =4δ[n+1]+4δ[n]+4δ[n]+4δ[n-1]
    =4δ[n+1]+8δ[n]+4δ[n-1]
y[n]={4,8,4}
```
